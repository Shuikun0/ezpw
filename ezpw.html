Let M = UTF-8(master), site′ = trim(site).toLowerCase(), account′ = trim(account).toLowerCase().
Let mode be “plain” (checkbox off) or “sym” (checkbox on).

Salt string (UTF-8):  ezpas-v6|site′|account′|mode
  mode = plain  → no special characters in any pool
  mode = sym    → SPECIAL and FULL include symbols below

Salt bytes:             SHA-256(salt string)
PBKDF2:                 HMAC-SHA256, password M, salt bytes, iterations 100000, dkLen 256 bits → B[0..31]

Pools (exact strings):
  UPPER   = ABCDEFGHIJKLMNOPQRSTUVWXYZ
  LOWER   = abcdefghijklmnopqrstuvwxyz
  DIGIT   = 0123456789
  SPECIAL = !@#$%^&*-_=+
  FULL_plain = UPPER + LOWER + DIGIT                    (length 62)
  FULL_sym   = UPPER + LOWER + DIGIT + SPECIAL          (length 76)

Byte cursor bi ← 0.
next() := v ← B[bi mod 32]; bi ← bi + 1; return v

If mode = plain:
  1) poolIds ← [0,1,2]. Fisher–Yates shuffle (i = 2 down to 1).
  2) pools ← [UPPER, LOWER, DIGIT]. For i = 0..2: chars[i] ← pools[poolIds[i]][next() mod |pool|].
  3) For q = 3..31: chars[q] ← FULL_plain[next() mod 62].

If mode = sym:
  1) poolIds ← [0,1,2,3]. Fisher–Yates shuffle (i = 3 down to 1).
  2) pools ← [UPPER, LOWER, DIGIT, SPECIAL]. For i = 0..3: chars[i] ← pools[poolIds[i]][next() mod |pool|].
  3) For q = 4..31: chars[q] ← FULL_sym[next() mod 76].

Output = chars[0..31] joined. Slider N (4–32) selects the prefix used for copy.
